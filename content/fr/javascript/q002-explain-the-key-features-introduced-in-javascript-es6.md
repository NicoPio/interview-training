# Expliquez les fonctionnalités clés introduites dans JavaScript ES6

Dans ES6, JavaScript a introduit ces fonctionnalités clés:

1. **Fonction fléchées**: Concise syntax for anonymous fonctions with lexical scoping.
2. **Template Literals**: Enables multiline Chaîne de caractèress and variable inclusion for improved readability.
3. **Destructuring Assignment**: Simplifies extraction of values from arrays or objects.
4. **Enhanced Object Literals**: Introduces shorthand notation for defining object methods and dynamic propriété names.
5. **Promises**: Streamlines asynchronous programming with a cleaner, structured approach.
