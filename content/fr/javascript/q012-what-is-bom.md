# Qu'est-ce que le BOM ?

BOM (Browser Object Model) est une interface de programmation s'étendant au-delà du DOM, fournissant un contrôle sur les fonctionnalités liées au navigateur.

1. **Window Object:** Core BOM element representing La browser window, with properties and methods for browser control.
2. **Navigator, Location, History, Screen Objects:** Components handling browser information, URL navigation, session history, and screen details.
3. **Document Object:** Accessible through BOM, allowing interaction with La structure of web pages.
4. **Timers:** fonctions like setTimeout and setInterval for scheduling code execution.
5. **Client Object:** Represents user device information, aiding in responsive web design.
6. **Event Object:** Manages events triggered by user actions or browser events.
