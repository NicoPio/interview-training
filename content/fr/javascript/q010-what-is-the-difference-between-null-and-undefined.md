# Quelle est la différence entre Null et Indéfini



| Caractéristique | Null | Indéfini |
| ------------------------ | --------------------------------------- | -------------------------------------------------- |
| Type | Object | Indéfini |
| Définition | Valeur d'affectation indiquant aucun objet | variable déclarée mais pas encore affectée une valeur |
| Nature | valeur primitive représentant Null/vide | valeur primitive utilisée quand une variable n'est pas affectée |
| Représentation | Absence d'une valeur pour une variable | Indique l'absence de la variable elle-même |
| Conversion dans les opérations | Converti en zéro (0) | Converti en NaN lors des opérations primitives |
